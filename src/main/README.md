# Final Project

This is a system that allows multiple different types of users to work and manage
features that you would find in a fitness center. Each type of user (Member, Trainer and Admin)
have different options available to them that allows them to have the smoothest interface
possible.

## How to Run
1. Clone the repo
2. Set up PostgreSQL with a database (name: FinalProject)
3. Edit the .xml file in the resources directory so it matches the database name, and your specific username and password
4. Run the main.java file and follow the menus provided. 
5.  Running the program for the first time should initialize a bit of default data for things that weren't specified (Like trainers, rooms and admins)
